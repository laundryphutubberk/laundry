# Responsibility Registry

Defines roles, responsibilities, required boot packages, handoff inputs/outputs, and ownership boundaries.

Core Boot -> Responsibility Boot -> Execution Boot -> Module Boot.

Roles:
- Chief Architect
- Data Architect
- Backend Architect
- Frontend Architect
- QA Architect
- DevOps Architect
- BE Worker
- FE Worker

Each role owns only its responsibility boot and consumes only required knowledge.
