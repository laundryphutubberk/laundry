# UI-ADAPTIVE-GUIDE.md

## Goal
ข้อมูลเหมือนกันทุกอุปกรณ์ เปลี่ยนเฉพาะ Layout

## Breakpoints
- Desktop >=1280
- Tablet 768-1279
- Mobile <768

## Rules
- One Component, Multiple Layouts
- ห้ามสร้าง Desktop/Mobile Component แยก
- Desktop ใช้ Sidebar
- Mobile ใช้ Drawer หรือ Bottom Navigation
- ทุกหน้าต้องเริ่มจาก 'งานที่ต้องทำ'

## Touch Rules
- ปุ่มใหญ่
- ใช้งานมือเดียวได้
- ลดการพิมพ์
- เน้น Scan / Tap มากกว่า Input
