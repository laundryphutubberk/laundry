# BOOT

## Boot Principle

Boot คือการทำให้ Human และ AI เข้าใจบทบาท ขอบเขต
และจุดเริ่มต้นของงาน ไม่ใช่การอ่านเอกสารทั้งหมด

เป้าหมายคือ Boot ให้สั้นที่สุด แต่ถูกต้องที่สุด

---

## Required Read Order

1. Project Constitution
2. Project Boot
3. Business Blueprint
4. Engineering Blueprint
5. Domain Model (schema.prisma)
6. Contracts
7. Execution Domain

---

## Boot Questions

ก่อนเริ่มงาน ต้องตอบได้ว่า

- Project นี้แก้ปัญหาอะไร
- ผู้ใช้หลักคือใคร
- Business Workflow คืออะไร
- Source of Truth คืออะไร
- กำลังเข้า Domain ไหน
- ขอบเขตของ Domain นี้คืออะไร
