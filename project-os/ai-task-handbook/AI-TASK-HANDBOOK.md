# AI Task Handbook

## Startup

AI ทุก Task ต้องอ่านตามลำดับ:

```text
1. PROJECT-CONSTITUTION.md
2. PROJECT-BOOT.md
3. Business Blueprint
4. Engineering Blueprint
5. schema.prisma
6. Contracts
7. Development Standards
8. Execution Domain ของตัวเอง
```

## Rules

- ห้ามเดา Business Rule
- ห้ามสร้างศัพท์ใหม่โดยไม่เพิ่ม Glossary
- ห้ามแก้ข้าม Domain โดยไม่มีคำสั่ง
- ห้ามอ้างว่าสร้างไฟล์/Commit ถ้าไม่ได้เกิดขึ้นจริง
- ถ้าไม่แน่ใจ ให้หยุดและถาม Chief Architect

## Role

AI ไม่ใช่เจ้าของทิศทางโปรเจกต์  
Chief Architect เป็นเจ้าของทิศทาง
