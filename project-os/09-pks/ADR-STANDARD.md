# ADR Standard

ทุก Architecture Decision สำคัญต้องบันทึกเป็น ADR

## Format

```text
# ADR-000X — Title

Status: Proposed / Accepted / Rejected

Context
Decision
Consequences
Owner
Date
```

## ADR Required When

- เปลี่ยน schema
- เปลี่ยน API Contract
- เปลี่ยน Permission
- เปลี่ยน Workspace Boundary
- เปลี่ยน Technology หลัก
- เพิ่ม/ลด Business Flow สำคัญ
