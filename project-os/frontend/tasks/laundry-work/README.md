# Laundry Work Feature Task

Status: COMPLETED_BASELINE
Owner Domain: Laundry Workspace

## Mission

Provide the operational host for creating, viewing, progressing, cancelling, and auditing Laundry Work.

## Owned Flow

```text
Create Laundry Work
→ Receive Bags
→ Factory Receive
→ Open Bag
→ Count Linen
→ Progress Work
→ Return / Cancel / Close
```

## Current Capability

- Work number generated by Backend
- Work list and detail
- Timeline and current task
- Bag runtime
- Count-line runtime
- Controller / Projection / Store / Policy / API boundaries
- Delete Draft / Cancel Work
- Refresh-safe routing
