# Frontend Task Boot Contract

## Purpose
Standard boot sequence for every Feature Task.

## Boot Sequence
1. Read project-os/BOOT-INDEX.md
2. Read project-os/frontend/tasks/README.md
3. Read project-os/frontend/tasks/TASK-INDEX.md
4. Locate the target feature task.
5. Read <feature>/README.md
6. Read <feature>/TASK.md
7. Read <feature>/STATUS.md
8. Load applicable FE-02 through FE-08 engineering standards.
9. Execute work.

## Supported Commands
- OPEN TASK
- CONTINUE TASK
- COMPLETE TASK
- REOPEN TASK

## Rules
- Feature Tasks own business flows.
- FE-02 through FE-08 own engineering standards.
- Do not skip STATUS.md.
- Do not mark COMPLETED without required validation evidence.
