# Definition of Done (DoD)

Feature ถือว่าเสร็จเมื่อครบเงื่อนไขนี้

```text
□ Implement เสร็จตาม Scope
□ ไม่ละเมิด Business Blueprint
□ ไม่ละเมิด schema.prisma
□ ไม่ละเมิด Contract
□ ผ่าน Business Rules
□ ผ่าน Workspace Isolation
□ ผ่าน Review Checklist
□ อัปเดตเอกสารที่เกี่ยวข้องแล้ว
□ QA / Manual Test ผ่าน
□ ไม่มี Architecture Drift
```
