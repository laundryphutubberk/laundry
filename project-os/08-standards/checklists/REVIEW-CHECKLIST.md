# PKS Review Checklist

## Business
- ตรงกับ Blueprint หรือไม่
- เพิ่ม Step ที่ไม่มีประโยชน์จริงหรือไม่

## Data
- ใช้ schema เป็นภาษากลางหรือไม่
- Source of Truth ชัดเจนหรือไม่

## Contract
- FE / BE / DB / QA เข้าใจตรงกันหรือไม่

## UI
- รองรับ Desktop / Tablet / Mobile หรือไม่
- เป็น Task-Oriented UI หรือไม่

## Security
- Workspace Isolation ถูกต้องหรือไม่
- Resort เห็นเฉพาะข้อมูลตัวเองหรือไม่
