# Task Return Contract Template

Status: Standard
Scope: Backend Execution Package
Owner: Backend Architecture

## Purpose

Use this template when returning work from a parallel backend task.

## Return Contract

```text
Task:
Phase:
Milestone:
Status:
Commit List:
Files Changed:
Verification Result:
Review Result:
Known Exceptions:
Blocked Items:
Ready For Merge:
Next Suggested Step:
```

## Result Values

Verification Result:

```text
PASS
PASS_WITH_NOTES
BLOCKED
```

Review Result:

```text
APPROVED
APPROVED_WITH_NOTES
CHANGES_REQUIRED
BLOCKED
```

Ready For Merge:

```text
YES
NO
WAITING_REVIEW
```
